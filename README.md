# ShabdLok

A fun word puzzle game for Hindi speakers. Spin the roulette wheel to pick letters and place them on a crossword-style grid to form words — think Scrabble meets a word wheel.

## How It Works

- A roulette wheel presents a set of letters for each level
- Select letters from the wheel to spell words
- Place words on the crossword grid to complete the level
- Bonus words earn extra points

## Features

- 50 handcrafted levels
- Crossword-style grid gameplay
- Bonus word discovery
- No ads, no tracking, no data collection — fully offline

## Privacy

ShabdLok collects no data. See [docs/privacy-policy.md](docs/privacy-policy.md) for details.
